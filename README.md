# AI Generated Portfolio Website

## Overview
This project is a personal portfolio website created entirely using generative AI tools without manually writing code.

The website was built using structured prompts and AI-assisted development tools to demonstrate the power of prompt engineering and AI-driven web development.

## Live Website
(https://code-journey-22.preview.emergentagent.com/)

## Key Features
- Personal portfolio layout
- Project showcase section
- Clean UI design
- Fully generated using AI prompts

## Tools Used
- Generative AI tools
- Prompt engineering
- Web design tools

## Learning Outcome
This project demonstrates my ability to:
- Use generative AI effectively
- Design websites using structured prompts
- Build functional projects without manual coding
