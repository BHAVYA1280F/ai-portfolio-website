# AI Generated Portfolio Website

## Overview
This project is a portfolio website created entirely using generative AI tools and structured prompts without manually writing code.

The goal of this project was to explore the capabilities of AI-assisted development and prompt engineering.

## Features
- Personal portfolio layout
- Project showcase section
- Clean UI design
- Built using AI prompts

## Technologies Used
- HTML
- CSS
- Generative AI tools

## Learning Outcome
This project demonstrates:
- AI assisted development
- Prompt engineering
- Rapid website prototyping
